# 🎉 BMS Event Website

A full-stack web application built for managing **college events** at **BMSCE**.  
The platform allows students to view upcoming events, register online, and organizers to manage event details.

---

## 🚀 Features
- 🏫 Event listing with details (title, date, description, venue)
- 📝 Simple event registration system
- 👩‍💻 Frontend built with **HTML, CSS, ReactJS**
- ⚙️ Backend powered by **Node.js + Express**
- 🗄️ Database integration for storing registrations
- 📱 Responsive design for mobile and desktop
